ケリー基準が最大化しているのは、単純な平均リターンではなく、長期の幾何平均成長率です。このページでは、まず幾何平均と算術平均の違いを図でつかみ、そのあとでボラティリティ・ドラッグ、ケリーの山型カーブ、σ の導出、ハーフケリーへ進みます。

ケリー基準 = 長期の複利成長率を最大化するレバレッジ
L* = (μ − r) / σ²

レバレッジは期待収益を押し上げますが、同時に値動きの荒さも増やします。複利で見ると、その2つの力がちょうど釣り合う場所があり、それがケリー最適です。

g(L) ≈ r + L(μ−r) − 0.5 × (Lσ)²
この近似式は、レバレッジで増える部分と、ボラティリティで削られる部分を同時に見ています。
最適レバレッジは「たくさん賭けるほど良い」ではなく、「増やす力と削る力がちょうどつり合う点」で決まります。

1. なぜ算術平均ではなく幾何平均を見るのか

長期投資で知りたいのは、各年の平均点というより、結局いくらになったかです。たとえば、毎年 +10% で増えるケースと、+30%, -10%, +30%, -10% と上下に振れるケースは、どちらも算術平均は 10% です。それでも最終資産は前者が 146.41、後者が 136.89 で同じではありません。

これは、算術平均が途中の上下動を十分に反映していないからです。値動きが荒いほど、見た目の平均リターンほどにはお金は増えません。だから長期の複利成長を考えるときは、算術平均ではなく幾何平均を見る方が自然です。

算術平均は見た目の平均リターン、幾何平均は複利で見た実際の増え方に近い平均です。

算術平均

(r₁ + r₂ + … + rₙ) / n

各年のリターンを単純に平均したものです。各年の平均点を見るには便利ですが、複利の最終結果をそのまま表すわけではありません。

幾何平均

[(1+r₁)(1+r₂)…(1+rₙ)]^(1/n) − 1

実際の最終資産額と同じ結果になる「毎年一定の成長率」です。長期投資の複利成長率を見るなら、こちらの方が本質に近いです。

3. ボラティリティが高いほど、幾何平均は下がりやすい

1

期待リターンが同じでも、値動きが荒いと複利は不利になる

同じ算術平均でも、上下に大きく揺れる経路は最終資産額が低くなりやすくなります。これがボラティリティ・ドラッグです。

2

レバレッジは期待収益を押し上げるが、ボラも同時に増やす

レバレッジを上げるほど、期待収益のプラス面も、値動きのマイナス面も大きくなります。

3

その差し引きの山頂がケリー最適になる

押し上げる力と削る力の差が最大になる場所が、長期の幾何平均成長率を最大化する点です。

長期投資では、平均リターンを大きく見せることよりも、ボラティリティで複利がどれだけ削られるかを見る方が重要です。

5. σ はどう導出されるのか

S&P500 の σ=17% という数字は感覚ではなく、価格データから作ります。ここでの σ は価格の散らばりではなく、リターンの散らばりです。

A

日次リターンを作る

rₜ = ln(Pₜ / Pₜ₋₁) のように、日ごとの変化率を並べます。価格そのものではなく、どれくらい動いたかを見るためです。

B

その標準偏差を取る

日次ログリターン列の標準偏差が日次ボラティリティです。σ は「どれくらいブレやすいか」を表しています。

C

年率化する

σ_year = σ_daily × √252 として年率にそろえます。営業日ベースなら 252 日を使うことが多いです。

σ の見方

σ = std(日次ログリターン) × √252

期間を 3 年にするか 10 年にするかで σ は変わりえます。固定された真実というより、どのデータ窓で測ったかに依存する入力値です。

「S&P500 の σ は 17%」という言い方の中身は、過去のリターンのブレを集計して年換算した数字です。

6. 比較は S&P500 / QQQ / SOX のベース資産に絞る

比較スペースは小さめにして、まず元資産の違いを見る形にしています。固定 3 倍 ETF の名前より先に、元資産の μ と σ の違いを押さえた方がケリーの話に直結します。

7. ハーフケリーがよく出てくる理由

フルケリー

理論上の幾何平均成長率を最大化します。前提が当たっているなら山頂そのものです。

ハーフケリー

成長率をかなり残しながら、レバレッジ由来の揺れを大きく抑えます。推定誤差にもやや強くなります。

クォーターケリー

さらに保守的です。モデルに自信がないときや、持ち続けやすさを重視するときの選択肢です。

理論上もっとも速く増える量と、現実に持ち続けやすい量は同じとは限りません。そこでフラクショナル・ケリーが出てきます。

幾何平均

長期投資で何倍になったかに対応する平均成長率です。ケリー基準が最大化しているのはこれです。

ボラティリティ

値動きが荒いほど、幾何平均は算術平均より下がりやすくなります。複利ではブレが重い意味を持ちます。

ケリー基準

期待収益を増やす力と、ボラティリティで削られる力の差が最大になるレバレッジを探しています。

S&P500 で 2 倍前後が筋が良いという話は、「S&P が特別だから」ではなく、期待リターンとボラティリティの組み合わせがそのあたりに山頂を作りやすいからです。
